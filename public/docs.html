HTTP API Docs

This is a list of endpoints that are available on the API.

  • /api/gh
  • /api/search

/api/gh

This endpoint redirects to the GitHub repo of the Minecraft Launcher.

GET /api/gh

/api/search

This endpoint searches for mods on Modrinth.

GET /api/search?q=string

q is a required parameter that specifies the search query.

The endpoint returns a JSON object containing the search results.

Example

GET /api/search?q=example

Try it out: /api/search?q=example
